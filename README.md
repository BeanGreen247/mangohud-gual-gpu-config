# mangohud-dual-gpu-config

~/.config/MangoHud/MangoHud.conf

To install, use one of the following wget commands

For one similar to MSI afterburner aka original layout

wget -O ~/.config/MangoHud/MangoHud.conf https://raw.githubusercontent.com/BeanGreen247/mangohud-dual-gpu-config/refs/heads/main/MangoHud.conf


For the horizontal version 

wget -O ~/.config/MangoHud/MangoHud.conf https://raw.githubusercontent.com/BeanGreen247/mangohud-dual-gpu-config/refs/heads/main/MangoHud-minimal.conf
